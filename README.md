## Node.js 开发Web Server博客
学习node.js, 不使用框架，开发一个 Web Server博客。

### 功能
#### blog 路由
- 获取博客列表
- 获取博客详情
- 新建一篇博客
- 更新一篇博客
- 删除一篇博客
#### user 路由
- 登录

## 总结
- nodejs 处理 http 请求的常用技能，postman 的使用 [√]
- nodejs 开发博客项目的接口(未连接数据库，未使用登录)[√]
- 理解为何要将 router 和 controller 分开 [√]
